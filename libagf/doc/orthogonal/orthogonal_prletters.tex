\section*{Research Highlights}

\vskip1pc

\fboxsep=6pt
\fbox{
\begin{minipage}{.95\textwidth}
\vskip1pc
\begin{itemize}

 \item An orthogonal coding matrix is optimal for error-correcting codes
	 when solving for multi-class from binary.

 \item Orthogonal coding matrices were tested on six different datasets and
	 shown to be more accurate than random codes for all six datasets
		for both the classes and the probabilities.

 \item An orthogonal coding matrix greatly simplifies the calculation of
	conditional probabilities making the calculation significantly faster.

  \item Orthogonal error-correcting-codes are simple and elegant and suggest
	  new directions in the search for
	  faster and more efficient multi-class classification methods.

\end{itemize}
\vskip1pc
\end{minipage}
}
